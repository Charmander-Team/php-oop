### TO ADD FUNCTIONAL DB

* To create DB
```sql
CREATE DATABASE oop_pokemon CHARACTER SET utf8mb4;
```
* To import DB 
```shell
$ mysql -uroot -p oop_pokemon < ./php-oop/Core/DB/oop_pokemon_dump_10juin.sql
```
